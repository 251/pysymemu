source code 

Build a solver intance. This is implemented using an external native solver via a subprocess. Everytime a new symbol or assertion is added a smtlibv2 command is sent to the solver. The actual state is also mantained in memory to be able to save and restore the state. The analisys may be saved to disk and continued after a while or forked in memory or even sent over the network.
