A map of memory.

It represents a convex chunk of memory with a start and an end address. It may be implemented as an actual file mapping or as a StringIO. It may have symbolic overlays. Some of it may be symbolic data.

>>>           ####SS#######S################SSS####
              ^                                    ^
            start                                 end

select(self, start, size)

source code 

Returns the selected map and a tuple with the remaining maps within this map.

Parameters:
  • start - the start address of the selected map.
  • size - the size of the selected map.
Returns: tuple
  • If the selected portion is not overlapped with this map:
    • None, (self, )
  • If the selected portion starts within this map but ends beyond this map:
    • map<start, self.end>, (map<self.start, start-1>, )
  • If the selected portion fits within this map:
    • map<start, start+size>, (map<self.start, start-1>, map<start+size+1, self.end>)
  • If just the last part of the selected portion is overlapped with this map:
    • map<self.start, start+size>, (map<start+size+1, self.end>, )
  • If this map is contained in the selected portion:
    • self, tuple()
