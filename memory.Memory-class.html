mmapFile(self, addr, size, perms, filename, offset=0)

source code 

Creates a new file mapping in the memory address space.

Parameters:
  • addr - the starting address (took as hint). If addr is 0 the first big enough chunk of memory will be selected as starting address.
  • size - the contents of a file mapping are initialized using size bytes starting at offset offset in the file filename.
  • perms - the access permissions to this memory.
  • filename - the pathname to the file to map.
  • offset - the contents of a file mapping are initialized using size bytes starting at offset offset in the file filename.
Returns: int
the starting address where the file was mapped.
Raises:
  • error -
    • "Address shall be concrete" if addr is not an integer number.
    • "Address too big" if addr goes beyond the limit of the memory.
    • "Map already used" if the piece of memory starting in addr and with length size isn't free.
